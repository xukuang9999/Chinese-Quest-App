# Chinese Quest 300

A static web app for Australian primary students who speak English and are learning beginner Chinese characters.

## Run

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.

## Data

`data.js` is generated by `tools/build-data.mjs`. The build script creates
`data/chinese-study.sqlite` first, then exports the browser-friendly dataset.

```bash
node tools/build-data.mjs
node tools/validate-data.mjs
```

The generated dataset keeps 300 beginner-friendly common Simplified Chinese
characters, 500 SQLite-backed common words, 3 word examples per character, and 3
sentence examples per character. It combines online Xinhua dictionary character
explanations and English glosses, Xinhua-style phrase data, common-character
stroke metadata, learner-friendly English definitions, and Hanzi Writer
stroke-order animation at runtime.

## Custom characters

Learners can add their own characters from the title page. Custom characters are
stored separately in browser `localStorage` under
`chinese-quest-custom-entries-v1`, then appended to the course at runtime. Do not
overwrite this key during app updates. The original 300 generated characters are
locked in the UI, and saved custom characters have no delete action.
